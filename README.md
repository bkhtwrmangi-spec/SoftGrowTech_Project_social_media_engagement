📌 Project 2 – Task 2 (SoftGrowTech Internship)
Overview
This project focuses on analyzing social media performance data to uncover insights into audience engagement. The goal was to calculate engagement and engagement rate, identify top-performing posts, and visualize results for better understanding of influencer and platform efficiency.

Key Steps
Created new metrics:

Engagement = Likes + Comments + Shares

Engagement Rate = Engagement ÷ Follower Count

Ranked posts by both raw engagement and engagement rate

Visualized top 10 posts using Python (Pandas, Seaborn, Matplotlib)

Highlighted trends in audience interaction and influencer tiers

Skills Applied
Data Cleaning & Transformation (Pandas)

Statistical Analysis

Data Visualization (Seaborn, Matplotlib)

Insight Generation & Reporting

Outcome
This task strengthened my ability to turn raw datasets into actionable insights, a valuable step in my SoftGrowTech Internship journey.
